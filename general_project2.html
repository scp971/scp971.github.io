Project 2: Benchmarking of Generative Artificial Intelligence for Additive Manufacturing

This project aims to identify an initial set of quantitative metrics for evaluating GenAI models and propose a framework for benchmarking based on these metrics.

Project Details

This project presents a comprehensive methodology for evaluating the capabilities of various existing GenAI tools (GPT-4o, GPT-4 turbo, o1, o-3 mini, Gemini, Llama 3) in addressing diverse AM-related tasks. We propose three categories of Quantitative metrics, namely agnostic, domain task, and problem task metrics that are used to benchmarking the GenAI tools.
