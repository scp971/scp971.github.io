Developing GenAI Model for AM

In order to explore and develop foundation models for AM, researchers should follow a structured approach. Below are the key steps involved:

1. Identifying the AM Task

Defining the problem scope is the first step. This includes recognizing challenges in AM that could benefit from AI-driven solutions.

Benchmarking Metrics (Qualitative)

2. Selecting a Base Model

Choosing the right AI model is essential. Options include Large Language Models (LLMs), Generative AI (GenAI), diffusion models, and tools like GPT-4o and DALL·E.

Benchmarking Tools

3. Assessing the Base Model

Evaluating the selected model’s initial performance, primarily through zero-shot prompting, before considering fine-tuning.

Benchmarking Models

4. Prompt Engineering

If zero-shot performance is insufficient, advanced prompt techniques like few-shot prompting, Chain-of-Thought (CoT), and Retrieval-Augmented Generation (RAG) are explored.

5. Fine-Tuning

If prompt engineering is insufficient, model fine-tuning with AM-specific datasets is required for improved accuracy and domain adaptation.
